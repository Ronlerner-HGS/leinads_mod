# SCP Foundation Mod (Fabric 1.21.1)

A Minecraft mod inspired by the scp universe

## Requiremnts Before installing the mod, make sure you have the following:

- ** Minecraft Java edition**
- **Fabric Loader for 1.21.1**
- **Fabric API** (matching 1.21.1)

---

## Installation (From Releases)

1. Go to the **Releases** page of this repository
2. Download the latest `.jar` file for **Minecraft 1.21.1**

---

## Installing Fabric

If you do not already have Fabric installed:

1. Download the Fabric installer from  
   https://fabricmc.net/
2. Open the installer.
3. Select:
   - **Minecraft Version:** `1.21.1`
   - **Loader Version:** Latest
4. Click **Install**.
5. Open the Minecraft Launcher and select the **Fabric 1.21.1** profile once to generate folders then close the game

---

## Installing the Mod

1. Locate your Minecraft **mods** folder:
   - **Windows:**  
     `%appdata%\.minecraft\mods`
   - **Linux:**  
     `~/.minecraft/mods`
   - **macOS:**  
     `~/Library/Application Support/minecraft/mods`

2. Place the downloaded mod `.jar` file into the `mods` folder.
3. Make sure **Fabric API** is also present in the `mods` folder.

---

## Running the Mod

1. Open the **Minecraft Launcher**
2. Select the **Fabric 1.21.1** profile
3. Click **Play**
4. Once in the main menu, go to **Mods** to verify the mod is loaded

---

## Troubleshooting

- **Game crashes on startup**
  - Make sure Fabric API is installed and matches Minecraft 1.21.1
  - Ensure no incompatible mods are installed
- **Mod not showing up**
  - Confirm the mod is inside the `mods` folder and not a subfolder
  - Verify you are launching the Fabric profile

---

## Notes

- This mod requires **Fabric** and will not work on Forge or NeoForge
- Always use the correct mod version for your Minecraft version
